## Sources Reviewed

For each programme, the following assets were reviewed:

- PMD v2 (Programme Metadata Document) — module list, module durations, format table
- Commercial Sheet v2 — session count, contact hours, self-study, total learning hours, CPD points
- CPD Package v2 — CPD rationale, completion criteria, sample agendas (Experience Edition)
- Facilitator Guide v2 — module-level scripts, activities, debriefs
- Workbook v2 — participant activities, deliverables, reflection templates

## Assumptions & Derivation Notes

1. Module names and durations are taken directly from the PMD v2 documents.
2. Session count and duration per session are taken directly from the PMD v2 / Commercial Sheet v2 / CPD Package v2 format tables.
3. Contact Hours, Total Learning Hours and CPD credits are taken directly from the PMD v2 / Commercial Sheet v2 / CPD Package v2 tables.
4. Individual Activities Hours = Total Learning Hours − Contact Hours. This matches the "Self-study / reflection" column in the Commercial Sheets and CPD Packages.
5. The module-to-session mapping for **Online Live** and **Onsite** is a proposed structure generated by fitting the actual module durations into the session durations stated in the PMD. It is not hard-coded in the source documents and must be approved before scheduling.
6. The **Experience Edition** session mapping is guided by the CPD Package sample agendas where provided (Day 1: welcome + Module 1 + evening reflection; Day 2: recap + Modules 2–6 + extended practice + closing).
