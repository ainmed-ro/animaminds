## Calm Response Card — pagina de referință

Folosește această pagină în orice situație tensionată.

### 4 pași sub presiune

1. **Pauză** — respiră 3 secunde înainte să vorbești.
2. **Etichetează** — ce simt? Ce nivel de intensitate? (1–5)
3. **Alege strategia** — în funcție de nivelul de intensitate.
4. **Răspunde cu calm** — spune clar, respectos, profesional.

### Niveluri de intensitate și strategii

| Nivel | Semne | Strategie |
|---|---|---|
| 1–2 Disconfort | Tensiune ușoară, nedumerire | Ascultă, reformulează, propune pasul următor |
| 3 Dezacord | Ton ridicat, emoții vizibile | Validează parțial, limitează subiectul, stabilește un alt moment |
| 4–5 Ostilitate | Țipăt, amenințări, agresivitate | Protejează siguranța, încheie conversația, anunță manager/HR/securitate |

### Când NU este programul potrivit

- [ ] Terapie, consiliere sau mediere profesională
- [ ] Conflicte grave, de lungă durată sau cu implicații legale
- [ ] Situații cu pericol fizic iminent

## 6. Bancă de fraze pe sector

### Educație
- „Văd că sunteți îngrijorat/ă. Să clarificăm împreună ce știm."
- „Dacă vorbim cu calm, putem găsi cea mai bună soluție pentru copil."

### Sănătate
- „Înțeleg că așteptați un răspuns. Vă explic ce știm acum."
- „Siguranța pacientului este prioritatea noastră comună."

### Administrație publică
- „Înțeleg că situația vă afectează. Vă arăt pașii legali disponibili."
- „Nu pot face o excepție, dar pot să vă redirecționez către colegul potrivit."

### Business
- „Nu este util să găsim vinovați. Să găsim soluția."
- „Acest ton nu ne ajută. Oprim 5 minute și revenim cu calm."

### ONG
- „Misiunea noastră comună este să protejăm beneficiarii. Să vedem ce putem face."
- „Nu pot aproba asta fără verificare, dar voi reveni cu un răspuns clar."

## 8. Referral Decision Tree

Pentru situația mea:

- [ ] Este cineva în pericol fizic? → Cheamă securitatea / 112.
- [ ] Situația este o încălcare de comportament (hărțuire, agresiune repetată)? → HR / management.
- [ ] Este un conflict între două părți care necesită rezoluție structurată? → Mediere.
- [ ] Persoana arată semne de criză sau traumă? → Consiliere / EAP.
- [ ] Nu sunt sigur/ă? → Consult manager / HR.
- [ ] Pot gestiona cu tehnicile din program? → Continuă și documentează.
