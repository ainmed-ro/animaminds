# AnimaMinds Portfolio Strategy

**Approved:** July 2026  
**Scope:** 5 core programs + Experience Editions delivery format

## 1. Portfolio architecture

| Program | Category | Strategic role |
|---|---|---|
| AI Fără Haos | AI & FUTURE SKILLS | Technology literacy and practical AI adoption without losing human control |
| Conversații care Contează | COMUNICARE & COLABORARE | Communication, trust and feedback that drive performance |
| Calm sub Presiune | CONFLICTE & REZILIENȚĂ PROFESIONALĂ | Conflict management, self-regulation and communication under pressure |
| Busola Deciziilor | LEADERSHIP & SCHIMBARE | Direction, clarity and decision-making for individuals, teams and organizations |
| Avantajul Uman | COMPETENȚE UMANE PENTRU VIITOR | Human-only competitive skills in an AI-augmented world |

## 2. Differentiation rules

- **No overlapping categories.** Each program has one primary category.
- **No generic wellbeing positioning.** Calm sub Presiune is explicitly about conflict and pressure, not burnout.
- **No abstract personal development.** Avantajul Uman is tied to future-of-work competitiveness.
- **No random images.** Each program card uses one image that directly supports the program promise.

## 3. Program card structure

Every program card on `/programe` must contain, in this order:

1. Category badge
2. Program title + symbol
3. Challenge / value proposition (description)
4. Target audience
5. Available formats
6. Status badge
7. Call to action

## 5. Audience taxonomy

Programs can be marketed to:

- Profesori / cadre didactice
- Manageri
- Lideri de echipă
- Specialiști
- Antreprenori
- Personal medical
- Administrație publică
- Echipe din companii
- Studenți și tineri
- Profesioniști în reconversie
- ONG-uri

The word **"Corporatiști"** is forbidden. Use "Manageri", "Lideri de echipă", "Specialiști" or "Echipe din companii" instead.

## 6. Delivery formats

All programs use the same standard line:

```text
Online · La sediul organizației · Open cohort · Experience Edition
```

## 7. Experience Editions

Experience Editions are immersive, off-site programs. They use outdoor and group imagery. They are not tied to a single program — they are a delivery format available for Busola Deciziilor, Calm sub Presiune and Avantajul Uman.

Primary Experience Edition image: `group-outdoor.jpg`

## 8. Messaging guardrails

- Focus on competence, not entertainment.
- Lead with the professional problem solved.
- Avoid retreat, vacation or party associations.
- Use CPD-ready, organization-friendly language.
- Never promise certification where it is not yet confirmed.
