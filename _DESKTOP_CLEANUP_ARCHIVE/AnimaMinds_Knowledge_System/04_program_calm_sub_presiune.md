# Calm sub Presiune — Program Brief

## Category
**CONFLICTE & REZILIENȚĂ PROFESIONALĂ**

## Positioning
Pauză. Respirație. Reconstrucție.

## Challenge solved
Profesioniștii din roluri cu contact uman intens sunt confruntați cu conversații tensionate, conflicte și situații cu miză ridicată. Programul oferă instrumente concrete pentru a răspunde cu claritate, calm și profesionalism.

## Approved description
"Un program practic pentru profesioniști care gestionează conversații dificile, tensiuni și situații cu miză ridicată. Participanții învață să răspundă cu claritate, calm și profesionalism atunci când presiunea crește."

## Target audience
Profesori · Manageri · Personal medical · Administrație publică · Echipe din companii

## Formats
Online · La sediul organizației · Open cohort · Experience Edition

## Key outcomes
- Gestionarea emoțiilor în situații tensionate
- Comunicare stabilă sub presiune
- Reducerea escaladării conflictelor
- Păstrarea profesionalismului în discuții dificile

## Key messages
- Conflictul nu este un eșec; lipsa gestionării lui este.
- Calmul se învață și se antrenează.
- Profesionalismul sub presiune este o competență.
