## Detailed changes

### Workbook v2.0

**What changed:**
- Added an **AI Quick Start one-pager** at the beginning with:
  - 60-second prompt structure
  - 5 pre-use questions
  - Clear "when NOT to use AI" checklist
- Added **sector-specific prompt examples** in Section 6 for:
  - Education (feedback for students)
  - Healthcare (patient preparation instructions)
  - Public Administration (community centre announcement)
  - Business (client follow-up email)
  - NGO (donor update)

**Why changed:**
- Review finding: participants need an immediate next-day reference.
- Review finding: blank prompt fields can create anxiety without examples.

**Review finding resolved:** Yes — priority actions 1 and 2.

---

### Facilitator Guide v2.0

**What changed:**
- Added **exact opening script** with welcome, promise, agenda and ground rules.
- Added **exact closing script** with follow-up preview and three reflection questions.
- Added **time checkpoints** inside each module script.
- Added **slide references** in brackets for each module section.
- Added **Appendix A: Ethical Dilemma Cards** with 6 scenario cards.
- Added **Appendix B: Sector Scenario Briefs** with 5 station briefs.

**Why changed:**
- Review finding: another trainer should be able to deliver the program without guessing.
- Review finding: referenced materials (cards and briefs) were not included in v1.

**Review finding resolved:** Yes — priority actions 3, 4, 5, 6, 7 and 8.

---

### Commercial Sheet v2.0

**What changed:**
- Added **dedicated Duration, Learning Hours and CPD Points table**.
- Added **Competencies Developed** table.
- Added **"What participants receive"** section listing workbook, quick-start, templates, checklists and follow-up emails.
- Added **"What participants can do the next day"** section with concrete actions.
- Added CPD-related FAQ item.

**Why changed:**
- Review finding: decision-makers need to see duration, hours, competencies and next-day value explicitly.
- Review finding: missing "what you receive" section.

**Review finding resolved:** Yes — priority actions 9, 10 and 11.

---

### CPD Package v2.0

**What changed:**
- Added **sample agendas** for Online, On-site and Experience Edition formats.
- Added **T0/T1/T2 evaluation instruments** including:
  - Pre-program confidence questionnaire (5 items)
  - Current AI use self-assessment
  - Satisfaction survey (5 items)
  - 30-day follow-up survey (5 items)
- Added **completion criteria** (attendance, deliverable, reflection, survey).
- Added **remediation policy** for incomplete participants.
- Added **certificate wording** template.

**Why changed:**
- Review finding: CPD package needed actual instruments and completion rules to be accreditation-ready.
- Review finding: agendas and certificate wording were missing.

**Review finding resolved:** Yes — priority actions 12, 13, 14 and 15.
